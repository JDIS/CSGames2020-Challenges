Défi Base de Données, Qualifications JDIS CSGames 2020

Ce challenge est divisé en 3 niveaux. Chaque niveau donne un flag lorsque réussi. Les niveaux sont en ordre croissant de difficulté, mais peuvent être fait dans n'importe quel ordre.

Chaque niveau contient un tableau pré-rempli. Le but du niveau est d'écrire la requête SQL engendrant les mêmes résultats que le tableau. Les noms de colonnes doivent être les mêmes que celles du résultat attendu, mais l'ordre des colonnes n'est pas important. Faites attention aux majuscules.
